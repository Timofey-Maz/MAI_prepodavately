Парсер расписания преподавателей МАИ

Скрипт для автоматического сбора данных о нагрузке преподавателей с сайта института МАИ с сохранением результатов в формате Excel.

Описание

Программа автоматически определяет текущий семестр (осенний или весенний), собирает данные со страниц преподавателей и формирует  таблицу нагрузки. Это позволяет быстро получить информацию о том, какие дисциплины, в каком объеме и для каких групп ведет каждый преподаватель.

Функции программы:

- Автоматический сбор ссылок на всех преподавателей с главной страницы кафедры.
- Определение семестра (осенний/весенний) на основе текущей даты.
- Подсчёт нагрузки: суммирование академических часов по каждой дисциплине.
- Сохранение в Excel: для каждого преподавателя создается отдельный `.xlsx` файл.
- Прогресс-бар: визуальное отображение хода считывания информации.
- Графические уведомления: всплывающие окна об успешном завершении или ошибке.

Установка
1. Клонируйте репозиторий

Откройте терминал и выполните:

```bash
git clone https://github.com/Timofey-Maz/MAI_prepodavately.git
cd MAI_prepodavately
```
2.  Создайте и активируйте виртуальное окружение:
    ```bash
    python -m venv venv
    source venv/bin/activate
    ```
3.  Установите необходимые библиотеки:
    ```bash
    pip install -r require.txt
    ```
4.  Запустите основной скрипт:
    ```bash
    python src/final.py
    ```
5. Структура проекта:
MAI_prepodavately/
├── src/                   # Исходный код
│   └── final.py           # Основной скрипт парсера
├── output/                # Папка с результатами (создается автоматически)
├── require.txt            # Список зависимостей
├── README.md              # Описание проекта (этот файл)
└── .gitignore             # Исключения для Git (виртуальное окружение, временные файлы)
6. Требования:
Python версии 3.8 и выше
Стабильное интернет-соединение
Доступ к сайту institutes.mai.ru
